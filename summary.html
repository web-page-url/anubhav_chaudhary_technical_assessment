Styled the various components into an appealing, unified design inspired by modern workflow tools.

Part 3

Text Node Logic Completed

Enhanced the Text node with dynamic layout and variable sensing capabilities.

  • Dynamic Resizing: Integrated logic to adjust node width and height based on user text input.
  • Variable Detection: Implemented Regex to detect {{ variable }} syntax and dynamically generate handles on the left side.
  • Distributed handles evenly on the node using percentage-based positioning logic.
Part 4

Backend Integration Completed

Built the bridge between the React frontend and FastAPI backend.

  • Frontend: Updated submit.js to send the pipeline graph (nodes and edges) to the backend.
  • Backend: Implemented /pipelines/parse to calculate num_nodes, num_edges, and verify if the graph is a DAG using Kahn's algorithm.
  • UX: Replaced browser alerts with a polished PipelineResultsModal to display backend responses.
Bonus

Additional "Run" Feature Completed

Implemented the end-to-end execution flow as described in the requirements.

Execution Flow Logic:

When the Run button is clicked:

  1. System identifies all Input Nodes.
  2. Prompts the user for values (e.g., "Please enter value for input_1").
  3. Submits data to the /pipelines/execute endpoint.
  4. Backend performs a Topological Sort and processes data through nodes (e.g., Transform Node turns "hello" into "HELLO").
